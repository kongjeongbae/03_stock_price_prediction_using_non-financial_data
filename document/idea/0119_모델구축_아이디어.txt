모델 구축 아이디어

0. 현재모델
=> 1년전 재무정보(값)를 표준화하고 이를 이용해 1년 후 주가 예측

1. <증감폭 값>을 이용
=> 말이 안됨 적어도 표준화해야 함

2. <증감폭 값을 표준화한 값>을 이용
=> 증감폭을 사용하는 것은 말이 안되는 듯.

3. <증감율 값>을 이용
=> 이걸로 해야할 듯.

4. <증감율 값을 표준화 한 값>을 이용
=> 이걸 사용해야 할 듯 했으나 증가율 값을 이용해야 할 듯
=> 증가율 값을 표준화하면 대기업과 성장주간의 차이가 사라짐
=> 성장주의 증가율은 클테지만 대기업의 증가율은 상대적으로 작을건데
=> 표준화해버리면 대기업의 증가율이 과대평가될듯
